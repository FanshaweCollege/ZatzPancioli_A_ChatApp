# ZatzPancioli_A_ChatApp

This project consists of an Interactive Chat Application that allows users to create a nickname and join a live chat room.

## Installation

NPM - [install](https://www.npmjs.com/get-npm)<br>
Clone the project [here](https://github.com/annapancioli/ZatzPancioli_A_ChatApp)<br>
Install dependencies with 'npm install'<br>
Run it locally on port 3000 with 'npm start'

## Built With

[Vue.js](https://vuejs.org/)<br>
[Node.js](https://nodejs.org/)<br>
[Socket.io](https://socket.io/)

## Deployment

The master branch is currently linked to [Heroku](https://chat-app-azp.herokuapp.com/)
